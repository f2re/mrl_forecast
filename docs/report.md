# Аналитический отчет: Наукастинг осадков с использованием Deep Learning

## Обзор
Данный отчет обосновывает выбор архитектуры ConvLSTM для краткосрочного прогнозирования осадков (наукастинга) на горизонт до 1 часа.

## Архитектура: ConvLSTM
ConvLSTM (Convolutional LSTM) сочетает в себе преимущества сверточных слоев (извлечение пространственных признаков) и слоев LSTM (моделирование временных зависимостей). В отличие от обычных LSTM, где веса являются полносвязными, в ConvLSTM используются сверточные операции, что позволяет сохранять пространственную структуру радарных данных.

## Предобработка данных
Для эффективного обучения модели рекомендуется:
1.  **Декодирование BUFR**: Преобразование данных из формата BUFR в регулярную сетку.
2.  **Клиппинг и масштабирование**: Ограничение значений (например, 0-70 dBZ) и нормализация в диапазон [0, 1].
3.  **Скользящее окно**: Формирование последовательностей кадров (например, 4 кадра истории -> 4 кадра прогноза).

## Метрики валидации
Для оценки качества прогноза предлагается использовать:
- **MSE (Mean Squared Error)**: Среднеквадратичная ошибка.
- **CSI (Critical Success Index)**: Критический индекс успеха (для оценки пороговых значений интенсивности).
- **SSIM (Structural Similarity Index)**: Индекс структурного сходства для оценки визуального качества.

## Рекомендации по развитию
1.  **Улучшенные архитектуры**: TrajGRU, PredRNN.
2.  **Генеративные модели**: FlowCast, LDCast для борьбы с размытием прогнозов на длинных горизонтах.
3.  **Контроль качества**: Улучшенная фильтрация шумов и артефактов радара.
