# Этап 9. Отдельный контур российских ДМРЛ

## Почему это отдельный этап

Текущий рабочий источник проекта - NOAA NEXRAD Level II из AWS. Российские
ДМРЛ имеют другие источники, форматы, правила доступа, метаданные и качество.
Нельзя объявлять поддержку ДМРЛ только на основании наличия BUFR-декодера.

## Цель

Подключить российские данные как отдельный проверяемый ingestion track, не
ломая NOAA-контур.

## Исследовательские задачи

1. Определить целевой источник:
   - официальный архив;
   - оперативный источник;
   - разрешенные условия использования;
   - историческая глубина;
   - стабильность доступа.
2. Получить реальные образцы:
   - BUFR;
   - ODIM HDF5;
   - при необходимости `.vol`.
3. Зафиксировать station catalog:
   - идентификаторы;
   - координаты;
   - высоту;
   - диапазон;
   - доступные продукты;
   - частоту обновления.
4. Определить минимальный общий продукт с NOAA:
   - lowest-elevation reflectivity;
   - либо CAPPI фиксированной высоты.
5. Проверить единицы, mask, nodata, шкалу dBZ и QC.

## Реализация

1. Создать отдельные адаптеры по форматам.
2. Не использовать fallback synthetic data.
3. Привести данные к контракту единого pipeline этапа 2.
4. Хранить source-specific metadata.
5. Добавить fixture каждого реально поддерживаемого формата.
6. Создать отдельный QC report для сравнения NOAA и ДМРЛ.
7. До доказательства совместимости не смешивать источники в одном обучении.

## Тесты

- Каждый формат декодируется на реальном fixture.
- География сверяется с эталонной картой источника.
- Timestamp и station metadata подтверждены.
- Преобразованный grid соответствует контракту pipeline.
- Неподдерживаемый descriptor дает явную ошибку.

## Риски

- Источники могут иметь ограничения доступа и лицензирования.
- Продукты разных радаров могут быть несопоставимы без дополнительной
  нормализации.
- Старые и региональные форматы могут потребовать отдельного набора BUFR-таблиц.

## Критерий завершения

Для каждого заявленного российского источника существуют документированный
доступ, реальные fixtures, тестируемый декодер, QC и визуальная сверка с
эталоном.
