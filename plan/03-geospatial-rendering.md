# Этап 3. География и радарное нанесение

## Проблема

Текущая визуализация зеркально переворачивает карту по север-юг и неверно
сопоставляет локальные километры с Web Mercator. Координаты поддерживаются только
для части станций.

## Цель

Сделать карту метеорологически и картографически проверяемой.

## Проектное решение

1. Строить radar grid в локальной азимутальной эквидистантной проекции AEQD с
   центром в координатах станции.
2. Хранить координаты радара из metadata исходного файла или проверенного
   каталога станций.
3. Для изображения использовать ориентацию, согласованную с grid:
   `origin='lower'` либо явно переворачивать массив один раз в pipeline.
4. Преобразовывать локальную AEQD-сетку в CRS подложки.
5. Рисовать mask прозрачным цветом отдельно от слабой отражаемости.

## Задачи

1. Удалить fallback координат `(0, 0)` для неизвестной станции.
2. Добавить проверяемый station catalog.
3. Исправить ориентацию осей и подписать направление север-юг.
4. Исправить расчет extent через трансформацию углов локальной сетки.
5. Проверить range rings: `50`, `100`, `150`, `200`, `250 км`.
6. Проверить подписи азимута: север `0°`, восток `90°`, юг `180°`, запад `270°`.
7. Добавить режим сравнения с рендером Py-ART.
8. Добавить легенду статусов: `no data`, слабая отражаемость, осадки.

## Изменяемые области

- `src/map_visualization.py`
- новый каталог станций или metadata helper
- тестовые fixtures

## Тесты

- Синтетический marker на севере отображается выше центра.
- Marker на востоке отображается справа.
- Радиус контрольного кольца соответствует расстоянию после преобразования CRS.
- Неизвестная станция вызывает ошибку, а не перенос данных в Атлантический океан.
- Snapshot before/after визуально подтверждает исправление.

## Ручная метеорологическая проверка

1. Выбрать реальный файл KOKX.
2. Отрисовать его в Py-ART или другом эталонном просмотрщике.
3. Сравнить положение основных ячеек, направление фронта и дальность.
4. Подписать UTC timestamp и координаты станции.
5. Сохранить эталонный скриншот в документации.

## Критерий завершения

Контрольные ячейки совпадают с эталоном, стороны света не перепутаны, кольца
дальности соответствуют локальной метрической геометрии.
