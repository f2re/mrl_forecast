# 🌦️ MRL Forecast Pro: Метеорологическая ИИ-платформа

Этот проект представляет собой профессиональную систему краткосрочного прогнозирования (наукастинга) метеорологических радиолокационных данных с использованием глубокого обучения (Convolutional LSTM). Платформа обеспечивает полный цикл работы с данными: от автоматического сбора архивов до обучения и применения ИИ-моделей.

## ✨ Ключевые возможности

- **🧠 Нейросетевой движок**: Многослойная сеть ConvLSTM на PyTorch для пространственно-временного прогнозирования.
- **🔄 Полный жизненный цикл (Lifecycle Management)**: 
  - Автоматизированный трекинг данных, датасетов и моделей с использованием метаданных.
  - Версионирование моделей в реестре с сохранением метрик и параметров обучения.
- **📡 Универсальные Дата-Адаптеры**:
  - **NOAA FTP (Live)**: Оперативное скачивание данных NEXRAD Level III.
  - **Archive Explorer**: Массовая загрузка данных NEXRAD Level II из облачных архивов AWS.
- **🖥️ Professional Web UI (Pro Dashboard)**:
  - **Инференс**: Выбор любой модели из реестра для получения прогноза.
  - **Менеджер данных**: Инвентаризация и управление загруженными архивами.
  - **Лаборатория моделей**: Запуск обучения с мониторингом логов в реальном времени.

## 📂 Структура проекта

```text
mrl_forecast/
├── scripts/                   # 🛠️ Скрипты автоматизации (Bash)
│   ├── setup.sh               # Установка и настройка окружения
│   ├── download.sh            # Скачивание архивных данных
│   ├── prepare.sh             # Подготовка датасета (Raw -> Numpy)
│   ├── train.sh               # Обучение модели
│   └── run_app.sh             # Запуск веб-платформы
├── src/                       # 🐍 Исходный код Python
│   ├── web_app.py             # Главный Flask веб-сервер (Backend)
│   ├── metadata_utils.py      # Утилиты управления метаданными и инвентарем
│   ├── train_nowcasting_model.py # Архитектура и обучение ConvLSTM
│   ├── make_dataset.py        # Пайплайн обработки радарных данных
│   └── ...                    # Декодеры и визуализация
├── templates/                 # 🎨 Современный Frontend (Inter UI)
├── data/                      # 💾 Хранилище (Raw & Processed)
└── models/                    # 🤖 Реестр обученных моделей (Registry)
```

## 🚀 Быстрый старт

1. **Настройте окружение**:
   ```bash
   bash scripts/setup.sh
   ```

2. **Запустите платформу**:
   ```bash
   bash scripts/run_app.sh
   ```
   Откройте браузер: `http://localhost:5005`.

## 🌐 Управление через веб-интерфейс (Рекомендуется)

Платформа MRL Forecast Pro позволяет управлять всем циклом без использования терминала:

1.  **Загрузка**: В «Менеджере данных» выберите станцию и дату. Система создаст уникальную папку с метаданными.
2.  **Подготовка**: Нажмите «Подготовить» у загруженного архива. Это преобразует сырые данные в обучающую выборку.
3.  **Обучение**: В «Лаборатории моделей» выберите датасет, настройте гиперпараметры и запустите обучение. Модель автоматически сохранится в реестр.
4.  **Прогноз**: На вкладке «Прогноз» выберите вашу лучшую модель из выпадающего списка.

## 🧬 Работа через терминал

Для продвинутых пользователей доступны Bash-интерфейсы:

### Шаг 1: Скачивание архива
```bash
bash scripts/download.sh KOKX 2024-05-20 100
```
*Данные сохраняются в `data/raw/archive/<ID_СЕССИИ>`.*

### Шаг 2: Создание датасета
```bash
bash scripts/prepare.sh 8 data/raw/archive/<ID_СЕССИИ>
```

### Шаг 3: Обучение модели
```bash
bash scripts/train.sh 20 4 1e-4 data/processed_archive/<ID_ДАТАСЕТА>
```
*Чекпоинты и метаданные сохраняются в `models/registry/`.*
